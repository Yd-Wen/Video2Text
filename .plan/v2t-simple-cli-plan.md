# Video2Text 简化版 CLI 项目规划

## 项目目标

构建一个轻量级、纯命令行的视频转文本工具，满足以下核心需求：
- **单命令运行**：一行命令完成视频到文本的转换
- **零依赖服务**：无需数据库、Web服务、消息队列
- **本地处理**：所有操作在本地完成，不上传数据
- **即装即用**：最小化配置，快速部署

## 性能优化策略

### 瓶颈点分析

| 环节 | 预期瓶颈 | 优化方向 |
|------|----------|----------|
| 音频提取 | FFmpeg 解码速度 | 降低采样率、使用硬件加速 |
| 模型加载 | 首次加载耗时 | 本地缓存、预加载提示 |
| 转录过程 | CPU/GPU 计算 | 模型量化、批处理 |
| 内存使用 | 长视频处理 | 分段处理、流式加载 |

### 具体优化方案

1. **音频预处理优化**
   - 提取时直接转为 16kHz 单声道（Whisper 要求）
   - 避免不必要的重采样
   - 使用有损压缩中间格式（如OGG）减少IO

2. **模型加载优化**
   - 显式指定模型缓存目录
   - 添加模型预下载命令选项
   - 支持模型路径自定义

3. **转录性能优化**
   - 使用 `fp16=False` 在纯CPU环境提升稳定性
   - 长视频启用 `vad_filter` 跳过静音段
   - 适当调整 `beam_size` 平衡速度和准确度

4. **内存优化**
   - 超过30分钟的视频启用分段处理
   - 每段重叠2秒避免断句
   - 使用生成器模式处理大文件

## 错误处理策略

### 需要覆盖的错误场景

| 错误类型 | 处理方式 |
|----------|----------|
| 输入文件不存在 | 清晰提示，退出码1 |
| FFmpeg 未安装 | 检测并提示安装方法 |
| 不支持的格式 | 捕获FFmpeg错误并提示 |
| 磁盘空间不足 | 预估所需空间并检查 |
| 模型下载失败 | 重试机制，代理配置提示 |
| 转录失败 | 保留临时文件供调试 |
| 输出目录不可写 | 前置检查并提示 |

### 退出码设计

```python
EXIT_SUCCESS = 0
EXIT_INVALID_ARGS = 1
EXIT_FILE_NOT_FOUND = 2
EXIT_FFMPEG_ERROR = 3
EXIT_TRANSCRIBE_ERROR = 4
EXIT_OUTPUT_ERROR = 5
EXIT_MODEL_ERROR = 6
```
